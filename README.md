# K4NAE — Portfólio

Meu portfólio pessoal, onde apresento meus projetos, tecnologias e formas de contato.

## Stack

- **React 19** + **TypeScript**
- **Tailwind CSS** — estilização
- **Framer Motion** — animações
- **Vite** — bundler
- **Formspree** — formulário de contato e newsletter

## Funcionalidades

- Tema claro/escuro com persistência em localStorage
- Loading screen com animação de digitação
- Cursor personalizado (dot + ring com lerp)
- Seções: Hero, Sobre, Tecnologias, Projetos, CTA, Contato, Footer
- Filtro de projetos por categoria
- Proteção anti-cópia e anti-clonagem
- SEO completo com Open Graph e Twitter Card
- Totalmente responsivo

## Rodando localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Configuração

Antes de publicar, atualize os seguintes arquivos:

- `public/protection.js` — troque `k4nae.dev` pelo seu domínio
- `index.html` — atualize `og:url`, `canonical` e `og:image`
- `public/cv.pdf` — substitua pelo seu currículo real
- `public/og-image.png` — adicione uma imagem de prévia (1200×630px)
